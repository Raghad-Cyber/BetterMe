BetterMe is a simple habit tracking web application built as a university group project.  
The system helps users build habits, track progress, and stay consistent using daily or weekly check-ins.

## Features
- User authentication (signup / login / logout)
- Create and manage habits
- Daily or weekly habit tracking
- Dashboard with progress overview
- Streak and progress calculation
- Simple and clean user interface

## Technologies Used
- PHP
- MySQL
- JavaScript
- HTML & CSS

## Setup Instructions
1. Clone the repository
2. Import the database using betterme_schema.sql
3. Create a config.php file by copying config.example.php
4. Edit config.php and set your database credentials
5. Run the project using XAMPP, WAMP, or any PHP local server

## Security Note
The file config.php is not included in this repository for security reasons.  
Use config.example.php as a template to create your own configuration file.
